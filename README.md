# Chidera Ojimba — Cloud Engineer Portfolio

My Personal portfolio website, a Cloud Engineer specialising in AWS, Azure, Kubernetes, Terraform, and CI/CD automation. Built with React, Vite, Tailwind CSS, and Framer Motion. Deployed on Vercel.

---

## Live Site

> https://derakings.vercel.app

---

## Tech Stack

| Layer      | Technology        |
| ---------- | ----------------- |
| Framework  | React 19 + Vite 7 |
| Styling    | Tailwind CSS 3    |
| Animations | Framer Motion     |
| Scroll     | react-scroll      |
| Linting    | ESLint 9          |
| Deployment | Vercel            |

---

## Features

- Animated hero section with tech stack badges
- Intro video section (YouTube embed)
- About section with profile photo and stats
- Skills grid covering cloud platforms, IaC, containers, CI/CD, monitoring, and more
- Projects showcase — 5 real-world cloud/DevOps projects with GitHub links
- Certifications section
- Contact section with email, LinkedIn, and GitHub links
- Custom SVG logo and favicon
- Fully responsive — mobile, tablet, and desktop
- Smooth scroll navigation
